# Flux - Caixa Inteligente

Sistema SaaS para gestão financeira e operacional de pequenos negócios prestadores de serviços.

## Tecnologias

- Java
- Spring Boot
- PostgreSQL
- React
- TypeScript
- Docker

## Estrutura

- `backend/` - API Spring Boot
- `frontend/` - Aplicação React
- `docs/` - Documentação
- `docker/` - Infraestrutura
- `scripts/` - Scripts auxiliares
